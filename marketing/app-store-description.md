## Full Description

**Learn IT and cybersecurity by DOING — not watching.**

NetQuest is a gamified learning app with 126 interactive missions that take you from absolute beginner to white hat hacker. No lectures. No slides. Just hands-on practice with a built-in terminal and hacking sandboxes.

━━━━━━━━━━━━━━━━━━━━
🎯 8 LEARNING PATHS
━━━━━━━━━━━━━━━━━━━━

🌐 NETWORKING
Master IP addresses, subnetting, DNS, ports, and protocols. Understand how data moves across the internet.

💻 TERMINAL
Conquer the command line. Navigate file systems, manage processes, write scripts, and automate tasks.

🐧 LINUX
Learn the OS that powers the internet. File permissions, package management, services, and system administration.

🐍 PYTHON
From variables to real tools. Build port scanners, password generators, and automation scripts.

🌍 WEB DEVELOPMENT
HTML, CSS, JavaScript — then go deeper into how the web actually works under the hood.

⚙️ DEVOPS
Git version control, Docker containers, CI/CD pipelines, and VPS server setup.

🔧 HARDWARE
What's inside your computer? CPUs, RAM, storage, networking hardware — understand the physical layer.

🛡️ CYBERSECURITY / WHITE HAT HACKING
The fun stuff. SQL injection, Cross-Site Scripting (XSS), password hash cracking, port scanning, CTF challenges, and building your own hacking home lab.
