## Full Description

**NetQuest is a gamified IT learning app with 126 hands-on missions that take you from "What is an IP address?" to SQL injection, XSS attacks, and building your own home lab.**

### The Problem
Learning IT is fragmented. YouTube videos are passive. Paid platforms cost $14-18/month and still rely on lectures. Nobody teaches the full stack — networking, coding, AND security — in one hands-on experience.

### The Solution
NetQuest is mission-based. Every mission is a specific, practical skill. You don't watch — you DO.

**8 Learning Paths:**
- 🌐 Networking — IPs, subnets, DNS, ports, protocols
- 💻 Terminal — Command line mastery
- 🐧 Linux — File systems, permissions, services
- 🐍 Python — From basics to real tools
- 🌍 Web Development — How the web actually works
- ⚙️ DevOps — Git, Docker, CI/CD, VPS
- 🔧 Hardware — Inside the machine
- 🛡️ Cybersecurity — SQL injection, XSS, hash cracking, CTFs

**Key Features:**
- Built-in terminal emulator — run real commands in the app
- Hacking sandboxes — practice exploits safely
- Progressive difficulty — each mission builds on the last
- Gamification — XP, completion tracking, path progression
- Affordable — free to start, $4.99/mo for full access

### Who It's For
- Complete IT beginners and career switchers
- Students supplementing coursework
- Anyone studying for CompTIA certifications
- People who learn by doing

### Pricing
Freemium. First missions in each path are free. Full access: $4.99/month — 3-4x cheaper than TryHackMe ($14/mo), HackTheBox ($18/mo), or Codecademy ($15/mo).

## Maker Comment

Hey Product Hunt! 👋

I'm John, the solo developer behind NetQuest. I built this because when I was learning IT, I couldn't find a single app that covered the full stack — networking, coding, and security — in a hands-on, interactive way.

Everything was either expensive, passive (video-based), or assumed you already knew the basics. I wanted something that could take a complete beginner and, mission by mission, build them up to doing SQL injection and building home labs.

126 missions later, here it is.

The app has a built-in terminal where you run real commands, hacking sandboxes where you practice SQL injection and XSS safely, and a gamified progression system that makes "one more mission" feel as addictive as "one more level."

I'm actively developing NetQuest and would love your feedback. What paths would you want to see next? What missions are missing? Your input directly shapes the roadmap.
