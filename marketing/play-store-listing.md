## Full Description (4000 chars max)

🚀 **Learn IT the fun way — by doing, not watching.**

NetQuest is a gamified learning app that teaches you real-world IT and cybersecurity skills through 126 interactive, hands-on missions. No boring lectures. No endless videos. Just you, a terminal, and challenges that actually teach you something.

**🎯 8 LEARNING PATHS**

🌐 **Networking** — IPs, ports, DNS, routing, firewalls, VPS deployment
💻 **Terminal** — Master the command line from zero to scripting
🐧 **Linux** — Users, file systems, services, administration
🐍 **Python** — Variables to APIs, build real tools
🕸️ **Web Dev** — HTML, CSS, JavaScript, responsive design, deployment
🔐 **Cybersecurity** — SQL injection, XSS, password cracking, penetration testing
🐳 **DevOps** — Git, Docker, CI/CD, Kubernetes, cloud infrastructure
🖥️ **Hardware** — CPUs, RAM, storage, build your own PC

**⚡ HOW IT WORKS**

Each path is broken into chapters with bite-sized missions. Complete missions to earn XP, level up, unlock achievements, and maintain your daily streak. Start as a "Noob" and work your way up to "Wizard."

**🎮 GAMIFIED LEARNING**
- 🔥 Daily streaks keep you motivated
- ⚡ XP and leveling system (7 ranks)
- ❤️ Lives system — learn from mistakes
- 🏆 Achievement badges for milestones
- 📊 Progress tracking across all paths

**🛡️ REAL CYBERSECURITY SKILLS**
- Practice SQL injection on safe, simulated targets
- Learn to scan ports with Nmap
- Understand XSS, CSRF, and web vulnerabilities
- Build your own hacking lab
- Capture the Flag (CTF) challenges

**💡 WHO IS THIS FOR?**
- Complete beginners curious about IT
- Students studying computer science
- Career changers moving into tech
- Anyone who wants to understand how the internet works
- Aspiring ethical hackers and penetration testers

**📱 WORKS OFFLINE**
Install as a PWA and learn anywhere — on the bus, in a waiting room, or during lunch break.

**🆓 FREE TO START**
The first 2 chapters of every path are completely free. Upgrade to PRO to unlock all 126 missions, get more lives, and access future content.

Download NetQuest and start your journey from Noob to Wizard today! 🧙‍♂️
