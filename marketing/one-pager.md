# NetQuest — Marketing One-Pager

---

## What Is NetQuest?

NetQuest is a gamified mobile app that teaches IT and cybersecurity through 126 hands-on missions. Users progress from absolute beginner concepts (what is an IP address?) to advanced skills (SQL injection, XSS attacks, building a home lab) — all through interactive, mission-based learning with a built-in terminal and hacking sandboxes.

## Key Stats

| Metric | Value |
|---|---|
| Total Missions | 126 |
| Learning Paths | 8 |
| Paths | Networking, Terminal, Linux, Python, Web Dev, DevOps, Hardware, Cybersecurity |
| Platform | Android (iOS planned) |
| Price | Free to start / $4.99/mo full access |
| Built-in Tools | Terminal emulator, SQL injection sandbox, XSS sandbox |

## Target Audience

**Primary:**
- IT beginners (18-35) with zero experience wanting to break into tech
- Career switchers from non-technical backgrounds
- CS/IT students who hate lecture-based learning

**Secondary:**
- CompTIA certification studiers (A+, Network+, Security+)
- Hobbyists curious about ethical hacking
- Self-taught developers wanting to fill knowledge gaps

## Unique Selling Points

1. **Full IT Stack in One App** — Networking, coding, DevOps, AND security. Competitors only cover 1-2 areas.
2. **Learn by Doing** — Built-in terminal and hacking sandboxes. No passive video content.
3. **Zero to Hacker Pipeline** — Starts from absolute zero and progressively builds to real exploitation techniques.
4. **3-4x Cheaper** — $4.99/mo vs $14-18/mo for competitors.
5. **Mobile-First** — Learn IT skills from your phone. No laptop required for most missions.
6. **126 Structured Missions** — Not a random collection of tutorials. A designed curriculum with clear progression.

## Revenue Model

**Freemium + Subscription**

- **Free Tier:** First several missions in each of the 8 paths (enough to demonstrate value)
- **Pro Subscription:** $4.99/month — unlocks all 126 missions, all paths, all sandboxes
- **Future:** Annual plan ($39.99/year), team/education licenses

## Key Metrics to Track

- Daily Active Users (DAU)
- Mission completion rate
- Free → paid conversion rate
- Path completion rate
- Day 1 / Day 7 / Day 30 retention
- Average missions per session

## The Vision

NetQuest aims to be the Duolingo of IT — the default first step for anyone who wants to learn technology, from networking to hacking. Mobile-first, hands-on, affordable, and comprehensive.
